# Retro Facilitator

**Activity:** Daily Operations  
**Tools:** Claude / ChatGPT / Gemini

## Prompt

```

- The problem (stated specifically)
- Root cause hypothesis (ask "why" 3 times)
- Suggested action (SMART: specific, measurable, assignable, realistic, time-bound)
```
